All-In-One creation guide

 

Install the features

  • unzip a Eclipse SDK 3.2 or higher (we have used 3.2.0 for KNIME 1.2.0)
  • (On Windows only: change the default workspace location
    • open the file ECLIPSE_ROOT/configuration/config.ini
    • set the property osgi.instance.area.default=workspace
    )
  • (On Linux only: Start eclipse always from commandline using the command line switch "-data workspace", this will prevent eclipse from writing a config file containing a workspace history, the windows trick mentioned above does not work)
  • install GEF
    • GEF SDK 3.2 can be found at SHARED/KNIME/KNIME_CDs/KNIME_1.2.0/GEF_SDK3.2
    • restart Eclipse after the installation if it was running
  • Copy the JRE to the eclipse directory, naming the folder "jre" (we use 1.5.0_11 for KNIME 1.2.0)
  • install the "org.knime.features.base" feature and the "org.knime.features.wizard" feature
    • Go to "Help" "Software Updates" "Find and Install" and enter the new update site "www.knime.org/update" and give it the name "KNIME" and the new update site "www.knime.org/update_development" and give it the name "KNIME Development", in the following select base and the wizard feature to be installed.

 

Init Perspectives

  • open the KNIME perspective, make sure it is visible in the upper right corner of the eclipse window, switch back to the java perspective and close the window

 

Add the README.txt file

  • copy the README.txt file into the Eclipse root directory 
    • The file can be found in the root directory of the "org.knime.product" plugin 

 

Add the documentation

  • create a directory "doc" in the Eclipse root directory
  • copy the whole PDF documentation from the web project "knime.org/WebContent/docs" to "doc"
  • check out the scripts project from svn and execute the javadoc target of the ant build.xml file
    • Create a directory within the doc directory called knime_javadoc and copy the previously created javadoc to this place

 

Zip the KNIME All-In-One

  • Before creating the zip the following things have to be done:
    • Delete the knime.log file in ECLIPSE_ROOT/workspace/.metadata/knime
    • Delete the install.log file in ECLIPSE_ROOT/configuration/org.eclipse.update
    • Delete the error_recovery.log file in ECLIPSE_ROOT/configuration/.settings
    • Remove the RECENT_WORKSPACES entry in file in ECLIPSE_ROOT/configuration/.settings/org.eclipse.ui.ide.prefs
  • finally create another zip (tar.gz) archive from the Eclipse root directory naming it "eclipse_knime_win32_ALL-in-ONE_1.2.0.zip" or "eclipse_knime_linux_ALL-in-ONE_1.2.0.tar.gz"
